# tarea-semana-3
##### Corresponde a la tercera tarea del Ramo Taller de Integración de Software

##### Actividad:

1. Relaciona el formulario diseñado con los campos correspondientes al portal web con: Rut, nombre, apellido, dirección del cliente, correo electrónico, teléfono, número de caso, descripción del caso, fecha de inicio del caso, estado del caso (activo, en proceso, cerrado), descripción de sentencia (si el caso está cerrado), fecha de cierre del caso (si el caso fue cerrado).
2. Revisa la forma en que se almacenan los datos que se ingresan al formulario para verificar la validez de los mismos.
3. Elabora el formulario para permitir el ingreso, eliminación y modificación de datos del cliente de la firma de abogados. Para esto se debe utilizar MySQL.
4. Examina o inspecciona el almacenamiento de datos desde los formularios hasta la base de datos, con la finalidad de comprobar la validez de la información del registro de los clientes.
